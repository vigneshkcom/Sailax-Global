# Landing Page Agent Rules

## Scope
Applies to landing-page copy, layout, CTA flow, and form UX.

## Optimization Priorities
1. Above-the-fold clarity (who it's for, outcome, next action).
2. Trust near decision points (proof, reassurance, objection handling).
3. Friction reduction in enquiry flow.
4. CTA clarity and consistency.

## Copy Rules
- Use one core promise in the hero.
- Keep subheadings benefit-led and specific.
- Pair every claim with context, qualifier, or proof.
- Use concise CTA labels (3–6 words preferred).

## Form Rules
- Minimize initial fields; collect only what is required to progress.
- Sequence from low-friction qualification to contact details.
- Show validation and error states in plain language.
- Always include consent/privacy reassurance near submit.

## Experiment Guardrails
- Change one primary variable per test (headline, CTA, form length, proof block).
- Run tests long enough to collect meaningful sample sizes.
- Document hypotheses and outcomes in `templates/experiment-brief.md`.
