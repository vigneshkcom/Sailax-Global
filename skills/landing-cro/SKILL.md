# Landing CRO Skill

## Audit Workflow
1. Identify the primary conversion goal and the primary CTA.
2. Score the page on clarity, trust, friction, and motivation.
3. Find the most likely bottleneck stage:
   - attention,
   - comprehension,
   - trust,
   - action.
4. Propose up to three high-impact changes.
5. Define event tracking needed to verify impact.

## Heuristics
- Prioritize message clarity above visual polish.
- Keep one dominant CTA intent per viewport.
- Place reassurance close to action points.
- Reduce field count and cognitive load before asking for contact details.
- Replace generic claims with specific numbers or concrete process detail.

## Output Format
Return recommendations in this structure:
1. **Diagnosis**: top bottleneck and evidence.
2. **Changes**: exact copy/UX updates in priority order.
3. **Measurement**: events and metrics to watch.
4. **Test Plan**: hypothesis, variant, success threshold, runtime.

## Definition of Done
- Recommendations are implementation-ready.
- Success criteria are measurable.
- Risks and rollback approach are clearly stated.
